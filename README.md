# Deploy Nanobot Instance Skill

Automatically deploy new nanobot instances via interactive Q&A.

## Install

Copy and run this in your nanobot chat:

```
[special:skill:install:https://github.com/yapex/new-nano-instance]
```

Or use the skill manager to add: `https://github.com/yapex/new-nano-instance`

## Usage

Trigger with:
- "部署新实例"
- "创建新 nanobot"
- "deploy nanobot instance"

See [SKILL.md](SKILL.md) and [REFERENCE.md](REFERENCE.md) for details.
